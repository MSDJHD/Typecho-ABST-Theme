# [Typecho-ABST-Theme](https://github.com/MSDJHD/Typecho-ABST-Theme/)

一款使用Bootstrap5制作的主题

## 使用

下载主题.zip包后，找到 **typecho所在文件夹/usr/themes**
将.zip包解压至此处即可
